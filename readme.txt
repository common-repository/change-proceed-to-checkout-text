Update default proceed to checkout button text without coding. Admin panel for updating the default WooCommerce checkout button text. 




== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on the 'Setting' option and then click on the Change proceed to checkout Text'.
